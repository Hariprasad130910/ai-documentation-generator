# AI Documentation Generator

## Project Overview

AI Documentation Generator is a simple web-based tool that converts Python source code into technical documentation automatically.

## Problem Statement

Developers often spend extra time writing documentation manually for code repositories. This process can be repetitive and time-consuming.

## Our Solution

This project allows users to upload a Python file and instantly generate structured documentation using AI-based prompt generation.

## Key Features

* Upload Python source code files
* Generate documentation automatically
* Simple and clean user interface
* 1-second loading indicator for better user experience

## Technologies Used

* Python
* GitHub
* Prompt-based AI generation
* Modern UI styling

## Benefits

* Saves developer time
* Improves code understanding
* Reduces manual documentation effort

## Future Scope

* Support multiple programming languages
* Repository-level documentation generation
* Export generated documentation
